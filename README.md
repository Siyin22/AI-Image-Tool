# AI 生图小工具

一个面向普通用户的本地 AI 图像生成小工具。它把 OpenAI 兼容接口、阿里云百炼/通义万相以及自定义中转接口整理成统一的图形界面，尽量让用户不用写代码、不用手动编辑配置文件，也能完成文生图、参考图生成和图库管理。

当前版本优先面向 Windows 本地使用。

## 功能特性

- 多配置管理：支持 OpenAI 兼容接口、阿里云百炼和自定义接口。
- 本地安全保存：配置、预设和图库历史保存在本机；API Key 使用当前 Windows 账号加密保存。
- 文生图：支持提示词、反向提示词、风格、比例、尺寸、数量和质量设置。
- 参考图生成：支持上传参考图，按模型能力调用图生图/编辑类接口。
- 百炼接入：支持百炼新旧图像接口，并对 `wan2.6+` 参考图能力做了实验接入。
- 任务控制：支持生成进度提示、任务详情、中止生成和失败重试。
- 图库管理：支持历史记录、搜索、复用参数、单张删除、批量删除和下载。
- 容错降级：主配置失败后，可以按优先级尝试备用配置。

## 快速开始

1. 安装 Node.js 18 或更高版本。
2. 双击项目目录里的 `start-tool.cmd`。
3. 浏览器会自动打开：

```text
http://127.0.0.1:17860
```

4. 使用期间请保持命令窗口打开。这个窗口负责本地配置读写、参考图临时上传和 API 请求转发。

如果 `17860` 已经有本工具在运行，启动脚本会直接打开已有页面。如果 `17860` 被其他程序占用，脚本会给出提示，此时可以关闭占用端口的程序，或修改 `start-tool.cmd` 中的 `PORT`。

## 首次配置

1. 点击右上角设置按钮进入配置中心。
2. 选择平台类型：OpenAI 兼容、阿里云百炼或自定义。
3. 填写 API Key、网关地址和模型名称。
4. 点击“保存配置”。
5. 点击顶部连接测试按钮，确认当前配置可用。

保存后，API Key 输入框会变为空，这是为了避免在界面上明文显示。再次保存配置时，API Key 留空表示继续使用已保存的 Key。

## 配置与数据保存位置

项目目录只保存工具代码，不保存你的 API Key。

本机数据默认保存在：

```text
%AppData%\AIImageTool
```

其中包括：

- `settings.json`：配置、预设和加密后的 API Key。
- `history.json`：图库历史记录。

在 Windows 上，API Key 会使用当前 Windows 账号加密保存。通常不需要额外输入密码；换浏览器使用时，也可以读取同一份本机配置。

## 生成图片

1. 输入主提示词。
2. 可选：上传参考图。
3. 选择风格、比例、尺寸、数量和质量。
4. 点击“生成”。
5. 图片会出现在最新结果区域，并自动进入图库。

生成过程中可以点击“中止”。中止会停止当前页面等待，并通知本地服务取消请求。对于已经提交给第三方平台并开始处理的任务，是否避免扣费取决于平台自身规则。

## 参考图说明

- OpenAI 兼容和自定义接口会在上传参考图后尝试调用图像编辑接口。
- 百炼 `wan2.6` 及更新模型已接入参考图实验支持。
- 单张参考图建议不超过 10MB。
- 当前最多上传 12 张参考图，总大小建议不超过 96MB。
- 参考图会先上传到本地临时目录，再由本地服务转发给模型接口。
- 如果平台返回字段或协议错误，可以先切回纯文生图，或改用 OpenAI 兼容接口完成图生图。

## 支持的平台类型

| 平台类型 | 说明 |
| --- | --- |
| OpenAI 兼容 | 适合 OpenAI 官方接口，以及兼容 OpenAI 图像接口格式的中转服务。 |
| 阿里云百炼 | 适合 DashScope/百炼图像生成接口，包含通义万相相关模型。 |
| 自定义 | 按 OpenAI 兼容模式尝试请求，适合自行部署或第三方网关。 |

百炼建议网关地址：

```text
https://dashscope.aliyuncs.com/api/v1
```

`wan2.6` 及更新模型会走新图像生成接口；`wanx2.1`、`wan2.5` 及更早模型会走旧通义万相接口。

## 常见问题

### 双击启动后提示端口被占用

如果提示本工具已经运行，直接访问：

```text
http://127.0.0.1:17860
```

如果提示端口被其他程序占用，可以关闭对应程序，或修改 `start-tool.cmd` 里的端口号。

### 换浏览器后配置还在吗

正常情况下还在。配置保存于本机 `%AppData%\AIImageTool`，不依赖某一个浏览器的 localStorage。

### API Key 会被上传到 GitHub 吗

不会自动上传。API Key 不保存在项目目录里，而是保存在 `%AppData%\AIImageTool`。只要不要把这个本机配置目录打包进仓库，就不会把 API Key 一起发布。

### 连接测试失败但生成可能成功吗

可能。部分 OpenAI 兼容中转站不支持 `/models` 之类的模型列表接口，导致连接测试失败，但生成接口仍然可用。

### 百炼 `wan2.7-image` 报模型或 URL 错误怎么办

可以先改成 `wan2.6-t2i` 验证账号、网关和 API Key 是否正常，再根据百炼后台实际开通的模型名称调整。
